Authentication Test Page

This is a simple test page to verify the application is running properly.

Steps to test:

  1. Go to http://localhost:3000
  2. Try logging in as admin user
  3. Check browser console for debug logs
  4. Try logging out and back in
  5. Verify role-based routing works

Expected behavior:

  • Admin users should see admin dashboard at /admin/dashboard
  • Employee users should see employee dashboard at /dashboard
  • Logout should work in both normal and private browsers
  • No infinite redirect loops

Note: Check the browser console for detailed debug logs to troubleshoot any issues.
